# Pildora-Limpieza-y-normalizacion-de-datos-Excel
Píldora formativa 20/04/2026: "Excel. Limpieza con fórmulas. Normalización de texto, fechas y uso de funciones lógicas."
En esta píldora formativa veremos la diferencia entre limpieza y normalización de texto en Excel, además de algunas acciones interesantes para estas tareas, tanto desde la barra de herramientas como a través de funciones integradas.
El objetivo es conseguir datos claros y fiables, para mejorar la calidad de los informes.
